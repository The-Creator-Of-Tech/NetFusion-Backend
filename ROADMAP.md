# NetFusion Platform Roadmap

This document outlines the strategic product vision and feature roadmap for future releases of NetFusion beyond Phase v1.0 RC1.

---

## Current Status: Phase v1.0.0-rc1 (Q3 2026) — ARCHITECTURE FROZEN
- 🟢 **Frozen Core Platform Architecture**: Decoupled modules, canonical data model, workflow case engine, multi-provider AI assistant.
- 🟢 **Collectors**: Sysmon EVTX, TShark PCAP, Nmap XML, Threat Intel APIs.
- 🟢 **QA & Verification**: 100% test pass rate, 6 real-world investigation scenarios, performance & security benchmarks.

---

## Phase v1.1.0 — Enterprise Scaling & Streaming (Q4 2026)
- 🔹 **Real-Time Kafka / Event Hub Ingestion**: Native streaming ingestion connector for high-throughput enterprise SIEM streams.
- 🔹 **Distributed Collector Runtime**: Sub-agent deployment for remote Windows/Linux host collectors with centralized orchestration.
- 🔹 **Advanced Graph Analytics**: Interactive Neo4j / NetworkX graph visualization for lateral movement and attack path analysis.

---

## Phase v1.2.0 — Autonomous SOAR & Fine-Tuned Local LLMs (Q1 2027)
- 🔹 **Bidirectional SOAR Actions**: Active response integrations with AWS Security Hub, CrowdStrike Falcon, and Palo Alto Panorama.
- 🔹 **Fine-Tuned Local Cyber LLMs**: Direct support for GGUF/vLLM quantized local cybersecurity models for air-gapped SOC environments.
- 🔹 **STIX 2.1 Native Export**: Automated threat intelligence sharing via TAXII/STIX protocol servers.

---

## Phase v2.0.0 — Next-Gen Multi-Agent Autonomous Defense (Q2 2027)
- 🔹 **Multi-Agent Collaborative Debating Engine**: Multi-agent adversarial reasoning for complex multi-stage APT investigations.
- 🔹 **Federated Multi-Tenant SOC Cloud**: Cloud-native SaaS architecture supporting multi-tenant SOC provider isolation.
